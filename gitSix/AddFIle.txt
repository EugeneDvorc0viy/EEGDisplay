Это вариант сырой программы
Основные функции
1. Можно создать эксперимент 
- После нажатия на кнопку "Создать новый" 
- В следующем окне нужно сначала ввести название теста
- Затем название задания, само задание и время, в течении которого нужно это выполнять
- После нажатия на кнопку "Добавить" элемент сохраниться в json файл, а поля очистяться
- Можно вводить произвольное количество заданий, они все сохраняются в тот же json файл
- Для перехода на Первый экран нужно нажать кнопку "Готово"
2. Можно просмотреть все Описания экспериментов
- Нужно выбрать его в БоксСписке сбоку и он сразу отобразится
3. Можно удалить описание эксперимента
- Нужно нажать на кнопку "Удалить" и ввести название удаляемой программы
4. Можно запустить программу на эксперимент
- Нужно нажать на кнопку "Провести эксперимент" (в шапке)
- Выбрать в БоксСписке нужный эксперимент, он отобразится на экране
- НУЖНО СОЗДАТЬ ДОП.ФАЙЛ сбоку под БоксСписком (на случай непредвиденного выхода)
- Как появилась надпись "Файл .. создан" нужно нажать "Начать"
- Каждое задание будет отображаться на экране
- Начнется отсчет времени для каждого задания, между ними программа будет засыпать на 3 секунды
* Программа будет генерировать случайные числа и записывать их как показания датчиков
- В конце эксперимента будет запрошено название для данного эксперимента
5. Можно посмотреть результат
- Нажать на кнопку "Базы Данных"
- Выбрать название требуемого эксперимента
- Нажать на кнопку "Показать", отобразится график эксперимента
